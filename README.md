这是一个帮助你导出 Xmind 导图中所有笔记内容的小工具。目前只有 Windows 版本。

类似下面的导图笔记，将其导出为 Markdown 的格式，保留链接。写这个工具是因为我做了这个导图，希望把参考资料也分享给我的小组，但是导图导出后无法查看笔记了，所以我寻求一个导出所有参考资料的方式。

使用方法也很简单：
1. 点击 exe 文件
2. 选择你要导出的 Xmind 文件
3. 导出完成，将在同目录下生成 文件名_notes 的 md 文件
